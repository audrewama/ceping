### 3. 银行风控措施
国际支付常见被拦截情形：
- 首次海外商户交易
- 额度异常变动
- 非工作时间操作

### 4. 账户资金问题
双重检查：
- 可用余额是否充足
- 交易冻结金额
- 单笔支付限额

- 账单地址准确性
- CVV安全码正确度

- 清除浏览器缓存
- 切换网络环境
